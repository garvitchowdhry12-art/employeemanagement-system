# Employee Management System

## Project Overview
This is a web-based Employee Management System developed using Core PHP, MySQL, and Bootstrap.

## Technology Stack
- PHP
- MySQL
- Bootstrap
- LAMPP/XAMPP

## Features
- User Login & Logout
- Role-Based Access Control
- Employee CRUD Operations
- Session Management

## How to Run
1. Start Apache & MySQL in XAMPP/LAMPP
2. Place project inside htdocs
3. Import employees_db.sql into phpMyAdmin
4. Open http://localhost/EmployeesManagement/login.php
